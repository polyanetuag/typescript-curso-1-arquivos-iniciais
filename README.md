## 📚 Sobre
Este projeto funciona como uma simulação para criar negociações.

Possui uma interface que possibilita criar negociações e listá-las em uma tabela, de acordo com as regras estabelecidas no projeto.

Tem como finalidade aprender os usos e recursos da ferramenta Typescript, comparando e substituindo o javascript já implementado.

## 🚀 Tecnologias utilizadas
- Javascript
- Typescript
- CSS

## ⏱️ Como executar

```bash
# Clonar o repositório
$ git clone https://github.com/polyanetuag/typescript-curso-1-arquivos-iniciais.git

# Entrar na pasta
$ cd typescript-curso-1-arquivos-iniciais

# Instalar as dependências
$ npm install

# Iniciar o servidor
$ npm start
```
